# Bhajan Gyan

A simple bhajan music player website.

## Project Structure

- `index.html` - main website page
- `style.css` - page styling
- `script.js` - music player logic
- `images/` - logo, cover images, background images, and GIFs
- `songs/` - MP3 song files

## GitHub Notes

Keep all image assets in `images/` and all audio files in `songs/` so the website paths stay organized.
